# Appointment-System
113上學期之軟體工程專案：指導教授預約系統（優化）

## 專案簡介
這是一個基於 Flask 框架開發的指導教授預約管理系統，主要用於中央大學資管系研究所學生與教授之間的預約面談管理。

## 主要功能
1. **預約管理**
   - 學生可以查看教授開放時段並進行預約
   - 上傳履歷檔案（PDF格式）
   - 查看預約狀態和歷史記錄

2. **教授功能**
   - 設定可預約時段
   - 審核學生預約請求
   - 管理預約記錄
   - 更新研究室資訊

3. **系統管理**
   - 帳號管理（學生、教授）
   - 預約狀態追蹤
   - 檔案管理

## 技術架構
- 後端：Python Flask
- 前端：HTML, CSS, JavaScript, Bootstrap
- 資料庫：MySQL
- 檔案存儲：本地檔案系統

## 系統角色
1. **學生**
   - 瀏覽教授資訊
   - 預約面談時段
   - 上傳履歷
   - 查看預約狀態

2. **教授**
   - 管理預約時段
   - 處理預約請求
   - 更新個人/研究室資訊
   - 查看學生資料

3. **系統管理員**
   - 帳號管理
   - 系統維護

## 預約狀態說明
- 待回覆 (status = 0)
- 已確認 (status = 1)
- 已拒絕 (status = 2)
